FRUIT STALL COIN SORTER — AUDIO MANIFEST (all optional)
========================================================
Drop MP3 files into this Audio/ folder with EXACTLY these names.
The game is silent for any missing file — nothing breaks.

SFX
  click.mp3           button press / dialogue advance
  card-whoosh.mp3     flashcard flies into the robot
  learn-pulse.mp3     robot absorbs a training example
  correct-chime.mp3   confident classification (soft, pleasant)
  reject-thud.mp3     not-coin rejection (SOFT — no harsh buzzer)
  unsure-boop.mp3     coin goes to the Unsure Box (curious, not sad)
  tray-drop.mp3       item lands in a tray
  celebrate.mp3       final test success / closing badge

VOICE-OVER (optional, add later)
  If you record narration for the dialogue lines, send them named
  vo-<number>.mp3 in spec order and Claude will wire each line to its
  speech bubble with a replay button.
